**This pack contains 5 mini games :dizzy: :sparkles:**


+ *brain-even*
+ *brain-calc*
+ *brain-gcd*
+ *brain-progression*
+ *brain-prime*


### Game brain-even
+ answer "yes" if the number is even and "no" if the number is odd
+ ***command brain-even***
